# Phase 08.1 Plan 01: Rebuild the Live Charting Shell

<objective>
Redesign the live charting screen so it matches the intended coaching workflow: zone first, matchup context always visible, controls sequenced clearly, and secondary history hidden.
</objective>

## Objective
- **What:** Replace the current bottom-dock layout with a top utility bar, a dominant zone workspace on the left, and a right-side control stack.
- **Why:** The current screen still allocates too much permanent space to secondary surfaces and does not reflect the scorer's real mental model.
- **Output:** A mode-ready live charting shell that maximizes the zone and keeps key context visible without clutter.
